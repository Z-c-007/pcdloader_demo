# BIN to PCD Converter

将 `.bin` 点云文件转换为 ASCII 格式的 `.pcd` 文件。

## 功能

- 批量处理指定文件夹中的所有 `.bin` 文件
- 将点云数据转换为 ASCII 格式的 PCD 文件
- 输出文件包含以下字段：`x`, `y`, `z`, `intensity`, `rgb`
- 根据点到中心的距离自动着色（距离越近颜色越亮，距离越远颜色越暗）

## 使用方法

### 命令行使用

```bash
node binToPcd.js <输入文件夹路径> <输出文件夹路径>
```

示例：
```bash
node binToPcd.js ./input ./output
```

### 使用 npm 脚本

```bash
npm run convert -- <输入文件夹路径> <输出文件夹路径>
```

示例：
```bash
npm run convert -- ./input ./output
```

## 数据格式说明

### 输入文件格式（.bin）

每个点由4个32位浮点数组成（共16字节）：
- x: 32位浮点数
- y: 32位浮点数
- z: 32位浮点数
- intensity: 32位浮点数

### 输出文件格式（.pcd）

ASCII 格式的 PCD 文件，包含以下字段：
- x: 点的X坐标
- y: 点的Y坐标
- z: 点的Z坐标
- intensity: 点的强度值
- rgb: 颜色值（根据距离中心的远近计算）

## 颜色映射规则

脚本会自动计算每个点到原点(0,0,0)的欧几里得距离，并根据距离远近映射颜色：
- 距离最近的点：颜色最亮（RGB值接近255,255,255）
- 距离最远的点：颜色最暗（RGB值接近0,0,0）
- 中间距离的点：根据距离比例映射灰度值
